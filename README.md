# BATIM
A web port of [Bendy and the Ink Machine](https://store.steampowered.com/app/622650/Bendy_and_the_Ink_Machine/), made by crackers.
If you use this on your own website/thing pls credit me!

# What's different in this port than the others?
- ALL chapters (1-5 & Archives)
- Fixed shaders
- Fixed lighting
- Fixed UI
- Fixed many more things
- Only ~600MB
- QoL additions
- And more!

# Known Issues
- Lag spikes in CH1
- Lag (not a lot of it but still)
- Minor shader Issues

## How To Locally Host
1. Clone/download this repo
2. Go to the repo's directory in CMD/terminal
3. Paste in:
```bash
npx serve
```
4. Download the serve node package (if it asks you to) with the (Y) key
5. Open the link it gives you
